# Lomakekomponentti

`<Form>` on React-komponentti, joka laajentaa Next.js:n [`<Form>`](https://nextjs.org/docs/app/api-reference/components/form)-komponenttia siten, että Next.js:n lomakekomponentin parannukset toimivat myös paikallistettujen polkunimien kanssa.

Kokeile seuraavaa esimerkkiä:

{/* prettier-ignore */}
<Form href="/form-component">
  <div className="flex flex-col gap-4 mb-8">
    <input
      type="text"
      name="q"
      placeholder="Hakukysely"
      className="bg-gray-200 text-gray-900 rounded-lg"
    />
    <div className="flex gap-2">
      <button
        type="submit"
        className="rounded-lg bg-gray-700 px-3 py-1 text-sm font-medium text-gray-100 no-underline hover:bg-gray-500 hover:text-white"
      >Lähetä</button>
      <button
        type="reset"
        className="rounded-lg bg-gray-700 px-3 py-1 text-sm font-medium text-gray-100 no-underline hover:bg-gray-500 hover:text-white"
      >Tyhjennä</button>
    </div>
  </div>
</Form>
{/* prettier-ignore-end */}
