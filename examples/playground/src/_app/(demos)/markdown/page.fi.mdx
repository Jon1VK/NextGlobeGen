# Markdown-sivut

NextGlobeGen tukee MDX-sivuja käyttämällä paikalliseen kieleen liittyvää tiedostonimeämiskäytäntöä.

Luo `page.<locale>.mdx`-tiedosto jokaiselle määritetylle kielelle reittisegmentin hakemistossa, josta markdown-sivun pitäisi olla saatavilla. NextGlobeGen käyttää kullekin kielelle oikeaa markdown-sivua tiedostonimen `<locale>`-osan perusteella.
