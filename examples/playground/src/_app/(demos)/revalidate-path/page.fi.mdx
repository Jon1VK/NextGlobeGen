# revalidatePath

`revalidatePath`-funktio laajentaa Next.js:n `revalidatePath`-funktion toimimaan lokalisoitujen reittien kanssa. Tämä mahdollistaa välimuistin tyhjentämisen tietyille lokalisoiduille sivuille datamutaatioiden jälkeen.
