# Kääntäjäfunktio

`useTranslations`-hook ja `createTranslator`-funktio palauttavat `t`-funktion. Sitä käytetään viestien kääntämiseen nykyiselle kielelle annetussa nimiavaruudussa.

Katso [ICU-dokumentaatio](https://unicode-org.github.io/icu/userguide/format_parse/messages/) ja seuraavat esimerkit käytettävissä olevista interpolointikuvioista.
